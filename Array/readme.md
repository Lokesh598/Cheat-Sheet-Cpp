## Array

Array is a collection of similer types of data elements. 
- how to define array in C++
- syntax: data_type array_name[array_size] 
```
int array[100];
```
- we can also initalize the array as follows


```
int array[3] = {1,2,3};
```
- how to define an empty array?

```
int array[3] = {};
